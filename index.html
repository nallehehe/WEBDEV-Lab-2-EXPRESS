Book Review API Documentation

Base URL: http://localhost:5000

Base URL: https://bookreviewing.azurewebsites.net

Create a Book Review

URL: /book

Method: POST

Response:

    Shows you the book review you just made.

Get Book Review by ID

URL: /books/{id}

Method: GET

Example:

        {
         "id": 5,
         "title": "Houseki no Kuni",
         "author": "Haruko Ichikawa",
    	 "review": "A captivating manga with beautiful artwork and an intriguing story about gemstone-based beings and their struggle for survival."
        }

Get All Book Reviews

URL: /books

Method: GET

Response:

    Shows you all the book reviews created and stored in the database.

Update A Book Review

URL: /books/{id}

Method: PUT

Response:

        {
            "id": [Updated id],
            "title": [Updated title],
            "author": [Updated author],
            "review": [Updated review]
        }

Delete A Bokk Review

URL: /books/{id}

Method: DEL

Response:

    No reponse to the user.
